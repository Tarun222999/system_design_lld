The Observer Pattern is a behavioral design pattern where:

One object (the Subject) holds state.
    Many objects (the Observers) want to know when that state changes.
    When state changes, the Subject notifies all observers automatically.
Core pieces
    Subject / Publisher
        Keeps a list of observers
        Methods like attach(), detach(), notifyObservers()
Observer / Subscriber
    Defines an update() method
    Gets called when the subject changes
